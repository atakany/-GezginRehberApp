# GezginRehber 📱🗺️

Bu proje, Mobil Programlama dersi kapsamında geliştirilmiş, kişiselleştirilmiş şehir rehberliği sağlayan mobil uygulama projesidir.

## 📌 Proje Özeti
GezginRehber, kullanıcıların bulunduğu konuma göre kişiselleştirilmiş mekan, etkinlik ve keşif önerileri sunan bir şehir rehberidir.

## 🧩 Özellikler
- Konum Tabanlı Mekan Önerisi
- Etkinlik Takvimi
- Kullanıcı Yorumları ve Puanlama
- Favoriler Listesi
- Google Maps Entegrasyonu
- Firebase Altyapısı

## 🎯 Hedef Kitle
- Gezginler
- Yerel Halk
- Genç Gezginler ve Öğrenciler


## 🔧 Kullanılan Teknolojiler
- Kotlin
- Android Studio
- Firebase
- Google Maps API
- Retrofit
- Glide
